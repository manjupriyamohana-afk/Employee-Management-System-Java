# Employee Management System (Java OOP)

This project is a console-based Employee Management System developed using Java. It demonstrates key Object-Oriented Programming concepts like inheritance, method overriding, and polymorphism.

## 🚀 Features
- Add Full-Time Employees
- Add Part-Time Employees
- Display Employee Details
- Salary Calculation
- Menu-driven system

## 🛠️ Concepts Used
- Inheritance
- Method Overriding
- Polymorphism
- ArrayList
- Scanner (User Input)

## ▶️ How to Run
1. Compile:
   javac EmployeeManagementSystem.java

2. Run:
   java EmployeeManagementSystem
